# 🚀 AITU Web Project

## 📌 Описание
Это учебный веб-проект, созданный с использованием Flask.  
Проект включает несколько страниц и систему задач (Task Manager).

---

## ⚙️ Функционал

- 🏠 Главная страница
- 📋 Система задач (CRUD)
  - Добавление задач
  - Удаление задач
  - Отметка как "done"
- 📊 Dashboard (статистика задач)
- 🖼 Галерея
- 📖 About страница
- 📞 Contact страница

---

## 🛠 Технологии

- Python 3
- Flask
- Flask SQLAlchemy
- SQLite
- HTML / CSS
- Jinja2 templates
